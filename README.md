* 데모앱: https://simseungyeol-todolist.netlify.app

<h1>기능</h1>

* 유저는 오늘 할일을 추가할수 있다.
 
* 체크기능과 삭제기능이 있다.
 
* 체크버튼을 누르면 할일이 끝난걸로 간주하고 글씨에 밑줄이 그어지며 회색으로 변한다.
 
* 삭제버튼을 누르면 할일이 리스트에서 삭제된다.
 
* 체크버튼을 누르고 한번 더 누르면 되돌리기 기능이 구현된다.
 
* 탭을 이용해 아이템들을 상태별로 나누어서 볼수있다.
 
* 모바일 버전도 확인 가능한 반응형 웹이다.


<h1>사용한 기술</h1>

* HTML

* CSS
